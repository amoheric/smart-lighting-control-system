# Design Decisions

## Sensor Placement

Motion sensors were placed centrally within each zone to maximize detection coverage while minimizing overlap. This ensures accurate occupancy detection across all functional areas.

## Daylight Sensors

Daylight sensors were positioned near perimeter areas to capture natural light variations, enabling dynamic dimming for energy efficiency.

## Fixture Layout

Lighting fixtures were evenly distributed across zones to provide balanced illumination and avoid dark spots.

## Control Panel Location

The control panel was centrally located in the hallway for ease of maintenance, reduced wiring complexity, and efficient coordination across zones.

## Zoning Strategy

The system was divided into functional zones (reception, conference, workspace, hallway, offices) to allow independent control while maintaining centralized logic.

## Network Integration

An IoT gateway and network layer were included to enable real-time monitoring and future scalability into smart building systems.
